# GitHub 开源快速参考

## 📝 一键复制命令

### 1. 创建GitHub仓库（在网页操作）
访问：https://github.com/new
- 仓库名：`shu-xiao-ban-math-tutor`
- 可见性：☑️ Public
- License：☑️ MIT License

### 2. 推送代码到GitHub（在终端执行）

```bash
# 进入项目目录
cd /workspace/projects

# 添加所有文件
git add .

# 提交代码
git commit -m "feat: 数小伴中职数学教学智能体首次开源"

# 添加远程仓库（替换your-username）
git remote add origin https://github.com/your-username/shu-xiao-ban-math-tutor.git

# 推送代码
git push -u origin main
```

### 3. 如果需要Token认证
访问：https://github.com/settings/tokens
- 点击 "Generate new token (classic)"
- 勾选 `repo` 权限
- 生成Token并复制
- 在git push时用Token代替密码

## ⚠️ 注意事项

1. **替换用户名**：将 `your-username` 替换成你的GitHub用户名
2. **仓库必须公开**：确保仓库是 Public，否则无法获得"开源分享"加分
3. **README要完整**：确保README.md显示正常
4. **包含LICENSE**：确保有LICENSE文件（MIT协议）
