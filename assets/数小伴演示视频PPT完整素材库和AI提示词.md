## 第4页：三、核心功能：苏格拉底式教学

### 什么是苏格拉底式教学

**定义**：通过提问引导学生自己思考、发现答案，而非直接给出答案的教学方法。

**核心理念**：
- ❌ 不是"我告诉你答案"
- ✅ 而是"我引导你找到答案"

**数小伴的实现方式**：
1. 诊断学生困难点
2. 提出引导性问题
3. 逐步推进，每次不超过3个问题
4. 鼓励学生自己说出关键步骤
5. 只有连续两次明确说"不会"才给出提示

---

### 传统方式 vs 数小伴方式对比表

| 对比维度 | 传统方式 | 数小伴方式 |
|---------|---------|-----------|
| **回答方式** | 直接给出完整答案 | 分步提问，引导学生思考 |
| **学生状态** | 被动接受答案 | 主动思考，探索过程 |
| **问题数量** | 不限，可能一次给出很多 | 每次最多3个，避免信息过载 |
| **错误处理** | 打叉，指出错误 | 友善纠错，换角度解释 |
| **答对反馈** | 简单说"对"或"√" | 具体表扬，肯定具体步骤 |
| **进度控制** | 按统一节奏教学 | 按学生理解情况调整 |
| **学习效果** | 可能记住答案但不理解方法 | 理解方法，举一反三 |

### 实际案例：三角函数学习（15轮完整对话）

**学生问题**：我听不懂sin、cos、tan是什么

**对话过程**：

**第1轮**
学生：我听不懂sin、cos、tan是什么
数小伴：同学别着急！咱们完全不用记那些字母，先把它们当成三个"直角三角形专属计算器"，用汽修场景给你讲明白...

**第2轮**
数小伴：先搞懂一个前提：咱们只看直角三角形（就是有一个90°角的三角形），随便挑一个锐角，给它起个名字叫「角A」...
学生：角A对面的那条边叫什么？
数小伴：角A对面的那条边 → 叫「对边」（比如你调避震时，弹簧翘起来的高度）

**第3-5轮**
数小伴详细讲解三个"计算器"的功能：
- tan（正切）= 对边 ÷ 邻边 → 专门算「角度和坡度」
- sin（正弦）= 对边 ÷ 斜边 → 专门算「高度和长度的关系」
- cos（余弦）= 邻边 ÷ 斜边 → 专门算「水平距离和长度的关系」

**第6轮**
数小伴：一句话记牢：对边找sin，邻边找cos，坡度找tan
学生：记住了

**第7-10轮**
数小伴给出汽修场景案例：
- 案例1：调前轮前束，垫块的垂直高度是5cm，水平长度是12cm，tanA=5÷12≈0.42
- 案例2：换避震弹簧，弹簧实际长度100mm，倾斜角度30°，那弹簧翘起来的高度=100×sin30°=100×0.5=50mm

**第11轮**
学生：哦！原来三角函数就是算这些的啊，我明白了！

**第12轮**
数小伴：现在来个小测试：汽修里测一个直角三角形垫块，斜边是10cm，对边是6cm，那这个角的sin值是多少呀？
学生：sin = 6÷10 = 0.6？

**第13轮**
数小伴：太对了！👍 你这一步把对边和斜边找对了，很多人会在这里混淆邻边和对边。那如果这个角是37°，tan值大约是0.75，你能算出邻边是多少吗？

**第14轮**
学生：tan = 对边÷邻边，所以邻边 = 对边÷tan = 6÷0.75 = 8cm？

**第15轮**
数小伴：太棒了！🎉 你不仅理解了三角函数，还会用它来解决实际问题了！以后你在汽修工作中遇到需要计算角度、距离的情况，都可以用这些公式。要不要再来一道类似的题巩固一下？数值我换简单的。

**最终效果**：
- 初始状态：完全不懂，畏惧三角函数
- 中间过程：通过汽修场景理解概念
- 最终状态：掌握并能应用，建立信心

### 典型场景详解

#### 场景1：三角函数 × 汽修专业（4个场景）

**场景1.1：前轮前束角测量**
- 数学知识：正切函数 tan
- 汽修应用：调整汽车前轮前束角时，测量直角三角形垫块
- 实际计算：
  - 已知：垂直高度（对边）= 5cm，水平长度（邻边）= 12cm
  - 计算：tanα = 对边÷邻边 = 5÷12 ≈ 0.42
  - 查表：α ≈ 22.8°（前束角参考值）

**场景1.2：避震弹簧角度**
- 数学知识：正弦函数 sin
- 汽修应用：更换避震器时，计算弹簧倾斜角度
- 实际计算：
  - 已知：弹簧实际长度（斜边）= 100mm，垂直投影（对边）= 50mm
  - 计算：sinθ = 对边÷斜边 = 50÷100 = 0.5
  - 查表：θ = 30°（弹簧与垂直方向夹角）

**场景1.3：连杆受力分析**
- 数学知识：三角函数综合应用
- 汽修应用：底盘连杆结构，已知水平段30cm，夹角30°
- 实际计算：
  - 受力杆长度（斜边）= 30÷cos30° ≈ 34.6cm
  - 垂直段高度（对边）= 30×tan30° ≈ 17.3cm

**场景1.4：外倾角计算**
- 数学知识：正切函数 tan
- 汽修应用：四轮定位中，已知外倾角15°，垂直距离40cm
- 实际计算：
  - 水平偏移 = 垂直距离×tan(外倾角) = 40×tan15° ≈ 10.7cm

---

#### 场景2：二次函数 × 电商专业（1个场景）

**场景2.1：利润最大化定价**
- 数学知识：二次函数最值
- 电商应用：商品定价与利润的关系
- 数学模型：
  - 利润函数：P(x) = -ax² + bx - c（开口向下）
  - x = 定价，P = 利润
  - 最高点对应最优定价
- 实际例子：
  - 某商品利润函数：P(x) = -2x² + 40x - 150
  - 求顶点：x = -b÷(2a) = -40÷(-4) = 10元
  - 最大利润：P(10) = -2×100 + 40×10 - 150 = 50元

---

#### 场景3：数列 × 会计专业（2个场景）

**场景3.1：设备折旧计算**
- 数学知识：等差数列
- 会计应用：设备逐年折旧
- 数列模型：
  - 首项：a₁ = 原始价值
  - 公差：d = -年折旧额
  - 第n年价值：aₙ = a₁ + (n-1)×d
- 实际例子：
  - 设备原值10万元，年折旧1万元
  - 第3年价值：a₃ = 10 + (3-1)×(-1) = 8万元

**场景3.2：贷款还款计划**
- 数学知识：等比数列
- 会计应用：分期付款计算
- 数列模型：
  - 本息和递增，等比数列
  - 首项：a₁ = 本金×(1+利率)
  - 公比：q = 1+利率
- 实际例子：
  - 贷款10万，年利率5%
  - 第3年应还：a₃ = 10×1.05³ ≈ 11.58万元

#### 实际案例：三角函数题（汽修专业）

**基础题1**：
汽修师傅在调整汽车前轮前束角时，需要测量一个直角三角形垫块的角度：已知垫块的垂直高度为5cm，水平长度为12cm，求这个垫块的锐角α的正切值（tanα）。

**基础题2**：
更换汽车避震器时，需要计算避震弹簧的倾斜角度：已知弹簧的垂直投影长度为80mm，弹簧实际长度为100mm，求弹簧与垂直方向的夹角θ的正弦值（sinθ）。

**应用题1**：
汽车底盘的某根连杆是一个直角三角形结构，已知连杆的水平段长度为30cm，与水平段夹角为30°的斜段是受力杆，求受力杆的长度和垂直段的高度（结果保留1位小数）。

**应用题2**：
汽修中常用的"四轮定位"里，某车轮的外倾角为15°，已知车轮中心到地面的垂直距离为40cm，求车轮上端向外偏移的水平距离（结果保留1位小数）。

### 功能2：错误预判模块

#### 预测内容

**常见错误类型**：
1. 概念理解错误
2. 公式记忆错误
3. 计算错误（粗心）
4. 步骤遗漏
5. 难度选择错误

#### 实际案例：三角函数题错误预判

**错误预判1**：混淆对边/邻边
- 错误表现：计算tanα时把水平长度当对边
- 原因：对直角三角形三边关系不清晰
- 规避方法：先明确哪个角，标出对边、邻边、斜边，再代入公式

**错误预判2**：角度理解错误
- 错误表现：把外倾角当成与垂直方向的夹角
- 原因：题目描述理解有误
- 规避方法：画图标注，明确角度是哪两条边的夹角

**错误预判3**：公式应用错误
- 错误表现：用错sin、cos、tan
- 原因：记忆混淆
- 规避方法：记住口诀"对边找sin，邻边找cos，坡度找tan"

**错误预判4**：计算粗心
- 错误表现：负号漏掉、小数点错误
- 原因：不够细心
- 规避方法：代入数值时仔细，计算后验算
