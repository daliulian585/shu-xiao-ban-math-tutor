## 📖 项目简介

数小伴是一款面向中等职业学校学生的数学教学智能体，基于豆包大模型和Coze智能体平台开发。通过苏格拉底式提问、专业场景连接和双模式设计，有效解决中职学生数学学习动力不足、基础薄弱、专业脱节等核心问题。

### 核心使命
- **愿意学数学**：通过生活化语言和专业场景激发学习兴趣
- **能学会数学**：采用苏格拉底式提问，分步引导降低认知门槛
- **知道数学有什么用**：将数学知识与汽修、电商、数控等专业紧密结合

---

## ✨ 功能特点

### 🎯 智能对话教学
- **诊断式教学**：自动识别学生问题类型（概念不清/公式记错/计算错误/完全没思路）
- **苏格拉底引导**：分步提问，不直接给答案，培养学生思维能力
- **生活化语言**：避免"显然""易得"等黑话，用学生能理解的表达

### 🔗 专业场景连接
- **系统化场景库**：覆盖6个专业、7个知识点、28个应用场景
- **实时关联**：识别数学知识点后，自动举专业应用例子
- **专业覆盖**：汽修、电商、数控、会计、建筑、旅游等

### 👨‍🏫 教师辅助模式
- **分层出题**：生成"基础-应用-挑战"三层作业（每层2题）
- **错误预判**：提供常见错误类型和规避建议
- **提问建议**：生成课堂提问引导

### 💝 情感与激励
- **具体表扬**：肯定学生的正确步骤
- **友好纠错**：不打叉，换角度引导
- **主动巩固**：每完成一个小环节，询问是否需要练习

### 📷 多模态支持
- **图片识别**：学生可拍照上传题目，自动识别数学内容
- **语音交互**：支持语音输入和输出（可选）

### 🧠 短期记忆
- **记忆窗口**：保留最近20轮对话（40条消息）
- **上下文连贯**：支持多轮对话和深度辅导

## 🛠 安装与部署

### 环境要求
- Python 3.12+
- Coze API Key
- 豆包大模型API访问权限

### 本地安装

1. **克隆仓库**
```bash
git clone https://github.com/your-username/shu-xiao-ban-math-tutor.git
cd shu-xiao-ban-math-tutor
```

2. **安装依赖**
```bash
# 使用 uv（推荐）
uv sync

# 或使用 pip
pip install -r requirements.txt
```

3. **配置环境变量**
```bash
# 创建 .env 文件
cp .env.example .env

# 编辑 .env 文件，填入你的API密钥
COZE_WORKLOAD_IDENTITY_API_KEY=your_api_key
COZE_INTEGRATION_MODEL_BASE_URL=https://ark.cn-beijing.volces.com/api/v3
```

4. **启动服务**
```bash
# 启动HTTP服务
bash scripts/http_run.sh

# 或启动流式服务
bash scripts/stream_run.sh
```

5. **访问服务**
```
http://localhost:8000
```

### Coze平台部署

1. 访问 [Coze智能体平台](https://www.coze.cn)
2. 创建新智能体，导入 `config/agent_llm_config.json` 配置
3. 配置Agent代码为 `src/agents/agent.py`
4. 发布并获取Web SDK地址

## 📖 使用方法

### 在线体验
直接访问 [数小伴Web界面](https://code.coze.cn/web-sdk/7630811434057760804)

### 学生使用

#### 1. 提问数学知识点
```
学生：我不理解函数的概念
数小伴：函数其实就像一个有特殊规则的机器...[详细解释]
```

#### 2. 上传题目
```
学生：[上传图片] 帮我解这道题
数小伴：这道题涉及一元二次不等式...[分步引导]
```

#### 3. 询问应用场景
```
学生：学三角函数有什么用？
数小伴：汽修里调前束角、建筑里算斜屋顶高度...[举例]
```

#### 4. 表达困难
```
学生：我听不懂
数小伴：那我们换个更简单的例子...[降低难度]
```

### 教师使用

#### 1. 生成分层作业
```
教师：我是老师，帮我出3道数列题，面向电商专业
数小伴：[生成基础、应用、挑战三层题目]
```

#### 2. 获取教学建议
```
教师：怎么讲二次函数最值？
数小伴：[提供教学策略、错误预判、提问建议]
```
